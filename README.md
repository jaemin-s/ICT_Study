통합 의학관리 시스템 입니다.
===

## 주요 기능
  1. 조회 기능 ( 처방 이력 조회, 의약품 조회 )
  2. 삽입 기능 ( 환자 정보, 처방전, 제품 정보 )
       - 처방전 작성시 환자의 처방중인 약 자동 출력
  3. 수정 기능 ( 제품 정보 수정 )
       - 제품 번호 입력시 제품 정보 출력
